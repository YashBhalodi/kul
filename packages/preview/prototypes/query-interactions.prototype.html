The converged model
Progressive disclosure, shallow
Click a person → anchored plate: detail on top, then two affordances. “Explore kin” opens one flat, scrollable list of every kin query with counts — no category nesting. Everything stays visible in place; nothing navigates away.
Relate, mouse-only, two doors
⚭ corner badges on every other card — one click, no mode. Or the plate’s “How related to…?” row, which arms a pick mode (dashed rings). Same resolution either way, with ⇄ swap in the banner.
Unchanged stances
Results glow + rest dims; ghosts light too; editor sync suspends during a query (Esc resumes); empty results toast honestly.
The relation card — six styles, two axes
Flip with the bottom bar or ← → keys while a card is open. Cards 1–3 vary shape: 1 Sentence (phrased + ⇄ swap), 2 Both ways (both directions, no swap), 3 Path chain (terms with their tree walk). Cards 4–6 vary density: 4 Whisper (term only — the tree outlines already say who; hover for the sentence), 5 One-liner (first names + terms, breadcrumb-style), 6 Dossier (both directions, blood/in-law tags, path chains, summary footer).
Language is a user setting
Phrasing renders in one language at a time — the header select stands in for a top-level user setting. In ગુજરાતી, hover a term for its transliteration; nothing is stacked bilingually.
